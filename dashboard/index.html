Quick Start

Up and Running in Seconds

Install SecProbe and start your first scan with a single command.

bash 
# Install SecProbe
$ pip install secprobe

# Run an audit scan with HTML report
$ secprobe https://target.com --mode audit --output html

# Full red team engagement with swarm coordination
$ secprobe https://target.com --mode redteam --swarm --federated
python 
# Programmatic usage
from secprobe import SecProbe

probe = SecProbe(
    target="https://target.com",
    mode="audit",
    divisions="all"
)

results = probe.run()
results.export("report.html")
API

REST API Endpoints

Integrate SecProbe into your CI/CD pipelines and security workflows via the API hosted on Railway.

POST
/api/scan
Initiate a new security scan against a target URL with specified mode and options.
GET
/api/scan/:id
Retrieve scan status and results by scan ID, including agent progress and findings.
GET
/api/scan/:id/report
Download the full scan report in HTML, PDF, or JSON format.
GET
/api/divisions
List all 20 divisions with their current status, agent counts, and capabilities.
POST
/api/scan/:id/stop
Gracefully terminate a running scan and collect partial results.
GET
/api/health
Health check endpoint returning system status, uptime, and agent availability.
